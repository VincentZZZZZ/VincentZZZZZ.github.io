尽管我们的手中空无一物

发表于 2017-10-10 | 分类于 胡思乱想

发现网易云方便的外链功能后还是迫不及待的想分享我最喜欢的这首歌曲，在网易云上也可以看这首歌的MV

关于这首歌曲

这首歌曲第一次听到时什么时候已经记不清了，只记得时网易云当时自动推荐的，第一次听到就被高潮部分给吸引住了，日文歌在节奏方面总是这么的让人欲罢不能。歌词我也很喜欢，我刚刚读大学，在这个学校与社会接轨的过渡期，真的感觉压力很大，自己手中一无所有，在硕大的校园里，也不像以前有着天天见面的朋友，仿佛空空荡荡的。但是怎么说呢，学到了东西总是开心地想与人分享；不知所措的时候也学会了坚持下去，迈出步伐。歌词真的很能引起共鸣，在这个情绪丰富而又孤独的年龄，每当看着繁星漫天，都会感觉自己手中空无一物，迷茫着，失去着。然而还是要拼命挣扎，在这不完美的世界，坚持心中的自己。我们都曾有梦想，心中住着理想中的自己，但是随着年龄增长，失去的不仅有时间，还有幻想和不羁。无论如何，还是希望在未来那迷茫之时，我能坚定地走下去，也不知道为了什么，但毕竟已经走了这么多年了吧。

关于hexo新版本配置问题

发表于 2017-10-09 | 分类于 杂项

hexo+GitHub配置的教程网上有很多，但是新版Hexo有几个地方会出错，在这里简单地总结一下

Deployer not found: github

在执行hexo deploy时，新版hexo会提示以上错误。这是因为新版部分配置文件要求进行了修改。下面是处理过程：

  1. 将_config.yml配置文件中的deploy下type字段后的github改为git

  2. 在Git Bash中执行npm install hexo-deployer-git --save(还有些版本中将save置后仍会出错，可以执行npm install --save hexo-deployer-git)

Please tell me who you are.
